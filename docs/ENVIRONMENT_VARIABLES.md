# Environment Variables

This document summarizes the current environment variable groups for the
Healthcare Backend.

## Core

- `NODE_ENV`
- `HOST`
- `PORT`
- `BASE_URL`
- `API_URL`
- `FRONTEND_URL`

## Database and Cache

- `DATABASE_URL`
- `DIRECT_URL`
- `REDIS_URL`
- `CACHE_PROVIDER`

## Security

- `JWT_SECRET`
- `SESSION_SECRET`
- `COOKIE_SECRET`

## Video

- `VIDEO_ENABLED`
- `VIDEO_PROVIDER`

## Integrations

- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
- `AWS_ACCESS_KEY_ID`
- `AWS_SECRET_ACCESS_KEY`
- `AWS_REGION`
- `FIREBASE_PROJECT_ID`
- `FIREBASE_PRIVATE_KEY`
- `FIREBASE_CLIENT_EMAIL`
